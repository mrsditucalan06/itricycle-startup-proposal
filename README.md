### Project Name: iTRICYCLE
**Type:** Tech Startup Proposal / Mobile App Concept

**Project Description:**
This is a project proposal it is for a safety application designed for students, staff, and residents of Mindanao State University-Main Campus. The app aims to help passengers track tricycles and Bajaj vehicles to prevent harassment, monitor unregistered vehicles, and ensure safe transportation around the campus.

**My Role & Contributions:**
1. Formulated the Business Model Canvas (BMC) for the startup idea.
2. Identified target customer segments and planned the cost structures.
3. Outlined how technology, smartphones, and campus security can work together to protect passengers.
